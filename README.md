# live-comment-ui
このプログラムは、ライブ配信風のコメントシミュレーションUIです。好きな動画をライブ配信中にしましょう。

## 📝 README: シンプルなライブコメントUIシミュレーション

このプログラムは、配信アプリなどでよく見られる、**シンプルで可愛らしいデザインのライブコメントの流れ**をシミュレーションするためのHTML、CSS、およびJavaScriptファイルです。

このUIは**グリーンバック**（`#66FF66`）でデザインされており、動画編集ソフトウェアと組み合わせることで、実際の配信映像の上に**「スマホ画面とコメント」**の部分だけを合成（クロマキー合成）して表示できます。

### 📌 特徴

* **完全な単一ファイル**: HTML、CSS、JavaScriptがすべて一つのファイルにまとまっており、扱いが簡単です。
* **アイコンレス**: アイコン画像の準備が不要な、名前とコメントだけのシンプルなデザインです。
* **速度調整可能**: コメントが流れる速度を簡単に変更できます。
* **クロマキー合成対応**: 背景が特定の緑色（グリーンバック）に設定されており、動画合成ツールでの利用を想定しています。

***

## ⚙️ プログラムの構成と使い方

### 1. ファイルの開き方

1.  このマークダウンの内容をすべてコピーし、**`index.html`**という名前で保存します。
2.  保存したファイルをWebブラウザ（Chrome、Edgeなど）で**ダブルクリック**して開きます。

→ 数秒後、画面中央にピンクのスマホが表示され、下からコメントが流れ始めます。

### 2. データの変更方法

プログラム内の `<script>` タグ部分を編集することで、表示されるコメントの内容を変更できます。

```javascript
const commentsData = [
    // 好きなコメントを追加してください
    { name: "アイン", text: "こんにちは" },
    { name: "ツヴァイ", text: "ありすちゃんきたあああああああ！！！！" },
    { name: "ドライ", text: "神降臨" },
    { name: "フィーア", text: "はえー" },
    // 新しいコメントを追加する場合
    { name: "新しい名前", text: "新しいコメント内容です" },
    // ...
];
